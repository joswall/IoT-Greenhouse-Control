# IoT-Greenhouse-Control
Remotely monitor greenhouse sensors and control heat, water flow, and cooling using ESP8266

**Sensors**

* High Sump Tank Level
* Low Sump Tank Level
* Return flow to fish tank
* Fish Tank Temp (F)
* Ambient Ligh Sensor
* Humidity
* Temperture (F)
* AC Feed Monitor
* VCC Backup Battery Monitor
* Solar Generation

**Control**

* Exhauast Fan On/Off
